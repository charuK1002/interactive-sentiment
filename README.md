# 🧠 Interactive Sentiment Analyzer

An interactive web application that analyzes the sentiment of user input text using Natural Language Processing (NLP).

This project allows users to enter any sentence or paragraph and instantly get the predicted sentiment as **Positive**, **Negative**, or **Neutral**.

---

## 🚀 Features

* Real-time sentiment prediction
* Clean and interactive user interface
* Fast NLP-based text analysis
* Instant feedback on user input
* Lightweight Flask backend

---

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/charuK1002/interactive-sentiment.git
```

Navigate to the project folder:

```
cd interactive-sentiment
```

Create a virtual environment:

```
python -m venv venv
```

Activate the environment:

```
venv\Scripts\activate
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

---

## 🌐 Usage

1. Open the web application in your browser
2. Enter a sentence or paragraph in the input field
3. Click **Analyze**
4. The model predicts the sentiment instantly
